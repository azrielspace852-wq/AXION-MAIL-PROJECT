Product Roadmap dan Strategi AXION MAIL:

### A. VISI & MISI
*   **Visi 1 Tahun:** Memvalidasi konsep MVP via Telegram Bot, mencapai 1.000 user aktif, dan menjadi gerbang SSO utama untuk XENON-V.
*   **Visi 3 Tahun:** Transisi mulus ke Web PWA, mencapai 100.000 user, dan menjadi standar identitas digital di ekosistem AXN.
*   **Visi 5 Tahun:** "Menjadi layanan email pribadi terbesar di Asia Tenggara dengan 5 juta user, terintegrasi penuh dengan seluruh ekosistem AXION."
*   **Misi:** 
    1. Memberikan akses email gratis yang aman untuk semua kalangan (dengan sistem *parent-child* untuk perlindungan anak).
    2. Menjadi gerbang login tunggal (SSO) yang mulus untuk semua aplikasi AXION (AXNDB, XENON-V, dll).
    3. Menjaga privasi dan kedaulatan data user sesuai prinsip AXN.
*   **USP (Unique Selling Proposition):** "Email gratis yang aman dengan SSO terintegrasi ke seluruh ekosistem AI dan produktivitas AXION."

### B. PRODUCT ROADMAP 12 BULAN (4 Fase)

#### 📅 FASE 1: MVP (Bulan 1-2) - "Buktikan Konsep"
*   **Tujuan:** Meluncurkan produk dasar yang berfungsi, memvalidasi pasar, dan mendapat 100 user pertama tanpa membebani HP Anda.
*   **Fitur Must Have:**
    ✅ Registrasi email @resend.dev (dengan validasi *parent-child* untuk < 12 tahun).
    ✅ Kirim & terima email (via Resend API).
    ✅ SSO token untuk login ke aplikasi ekosistem AXN.
    ✅ Backup otomatis ke Telegram Channel Private.
    ✅ Feedback user langsung ke grup Telegram AXION MAIL.
*   **Fitur Should Have:**
    ⭐ Inbox dengan pagination sederhana di Bot.
    ⭐ Profil user (lihat data sendiri).
*   **Fitur Could Have:**
    🔸 Multi-bahasa dasar (Indonesia, Inggris).
*   **Fitur Won't Have (Blokir demi kecepatan rilis 14 hari):**
    ❌ Folder/label, Spam filter, Template, Attachment, Dashboard web.
*   **Target:** 100 user aktif.
*   **Metrik Sukses:** User bisa registrasi dan kirim email pertama < 5 menit via Telegram.

#### 📅 FASE 2: GROWTH (Bulan 3-4) - "Buat User Betah"
*   **Tujuan:** Menambah fitur esensial agar user betah, mengurangi *churn rate*, dan memicu *word-of-mouth*.
*   **Fitur Must Have:**
    ✅ Dashboard web PWA (Akses ringan dari browser HP user).
    ✅ Spam filter sederhana (berbasis aturan kata kasar & domain mencurigakan).
    ✅ Attachment (max 10MB) menggunakan storage gratis.
    ✅ Multi-bahasa (Ekspansi ke 10 bahasa target pasar).
*   **Fitur Should Have:**
    ⭐ Auto-reply AI (Balas otomatis cerdas menggunakan Gemini).
    ⭐ Label/Category sederhana (Penting, Kerja, Pribadi).
*   **Fitur Could Have:**
    🔸 Dark mode di PWA.
*   **Target:** 1.000 user aktif.
*   **Metrik Sukses:** 50% user login > 3x/minggu.

#### 📅 FASE 3: MONETIZATION (Bulan 5-6) - "Hasilkan Pendapatan"
*   **Tujuan:** Mulai menghasilkan pendapatan (MRR) untuk menutupi biaya operasional tanpa mengganggu pengalaman user *free*.
*   **Fitur Must Have:**
    ✅ Paket Premium (Storage lebih besar, custom domain, akses AI Gemini tingkat lanjut).
    ✅ Integrasi pembayaran global (Stripe / Payment Gateway lokal).
    ✅ Integrasi SSO erat dengan AXN Command & XENON-V (Cross-selling).
*   **Fitur Should Have:**
    ⭐ Email scheduling (Jadwalkan kirim nanti).
    ⭐ Email templates (5 template siap pakai untuk profesional).
*   **Fitur Could Have:**
    🔸 Advanced analytics (Statistik pembacaan email).
*   **Target:** 5.000 user total, 100 premium user.
*   **Metrik Sukses:** Pendapatan $500/bulan, conversion rate > 2%.

#### 📅 FASE 4: EXPANSION (Bulan 7-12) - "Jadi Pemain Utama"
*   **Tujuan:** Skala besar, ekspansi pasar B2B, dan menjadi pemain utama di Asia Tenggara.
*   **Fitur Must Have:**
    ✅ Mobile App (React Native - iOS & Android).
    ✅ Email marketing tools (Kirim newsletter massal).
    ✅ API publik untuk developer (B2B Monetization).
    ✅ Collaboration (Shared inbox untuk tim/UMKM).
*   **Fitur Should Have:**
    ⭐ Encrypted email (End-to-end encryption untuk privasi ekstra).
    ⭐ Calendar integration.
*   **Fitur Could Have:**
    🔸 AI email composer (Tulis email otomatis dari prompt singkat).
*   **Target:** 50.000 user, 1.000 premium user.
*   **Metrik Sukses:** Pendapatan $5.000/bulan, user aktif di 10 negara.

### C. PRIORITAS FITUR (MoSCoW) UNTUK SEMUA FASE
*   **Must Have (Wajib):** Registrasi, Email dasar, SSO, Backup, Dashboard PWA, Spam filter, Attachment, Premium, Mobile app. *(Ini adalah nyawa produk).*
*   **Should Have (Sebaiknya):** Auto-reply AI, Label, Scheduling, Templates, Analytics, Encryption, Calendar. *(Ini yang membuat kita unggul dari kompetitor).*
*   **Could Have (Boleh):** Dark mode, AI composer. *(Hanya dikerjakan jika ada waktu luang / *resource* berlebih).*
*   **Won't Have (Tidak):** Social media integration, fitur berat yang memakan *resource* server gratis secara berlebihan.

### D. REKOMENDASI STRATEGIS (Khusus Solo Founder & HP)
1.  **Operasional via HP (Admin Panel Telegram):** Karena Anda bekerja solo dari HP, pertahankan Telegram Bot sebagai *Command Center* Anda di Fase 1. Anda bisa memonitor user, membaca feedback, dan membalas tiket langsung dari genggaman tangan tanpa perlu laptop berat.
2.  **Jaga Limit Free Tier:** Resend, Firebase, dan Cloudflare memiliki batas kuota gratis. Pastikan fitur *Rate Limiting* (maksimal request per menit) dan batas pengiriman harian (misal: 50 email/hari per user) diterapkan sejak MVP agar tidak ada user jahil yang menghabiskan kuota gratis kita.
3.  **Tunda Mobile App:** Jangan buat native mobile app sebelum user > 10.000. Mengelola *build* iOS/Android dari HP sangat melelahkan dan memakan waktu. Fokus pada PWA (Progressive Web App) yang bisa di-*install* seperti aplikasi native.
4.  **Fokus Komunitas:** Gunakan grup Telegram sejak MVP. Jadikan 100 user pertama sebagai "Beta Tester" yang merasa dihargai. Feedback mereka adalah QA (Quality Assurance) gratis untuk Anda.
5.  **Strategi Harga:** Harga premium $3/bulan atau $30/tahun sangat kompetitif dan terjangkau untuk pasar Asia Tenggara (pelajar/mahasiswa) dibandingkan Gmail/ProtonMail.

### E. RISK & MITIGASI (Dari Sisi Produk)
*   **Risk 1: User tidak suka UI/UX Bot Telegram.**
    *   *Mitigasi:* Gunakan *interactive buttons* dan *inline keyboards* di Telegram agar navigasi terasa seperti aplikasi, bukan sekadar *chat* teks. Lakukan survei kepuasan di akhir bulan ke-1.
*   **Risk 2: Kompetitor Raksasa (Gmail, Outlook).**
    *   *Mitigasi:* Jangan lawan mereka di fitur. Fokus pada *niche* kita: Ekosistem AXN (SSO ke XENON-V), perlindungan anak (*parent-child*), dan pasar Asia Tenggara.
*   **Risk 3: User tidak mau bayar (Konversi rendah).**
    *   *Mitigasi:* Tawarkan *free trial* 7 hari untuk fitur Premium, atau berikan kuota token XENON-V gratis jika mereka berlangganan AXION MAIL Premium (Bundling ekosistem).
*   **Risk 4: Kuota Infrastruktur Gratis Jebol.**
    *   *Mitigasi:* Buat notifikasi otomatis ke HP Anda via Telegram Bot jika penggunaan API Resend atau Firebase mencapai 80% dari batas harian/bulanan.
